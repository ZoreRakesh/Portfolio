Professional Journey

Key milestones and achievements throughout my career in SAP ABAP consulting.

Associate Consultant
Bristlecone | Mumbai, India
June 2024 – Present
Project: Harmony
Designing, developing, and maintaining 20+ ABAP solutions across S/4HANA — including ALV reports, RFC interfaces, RAP applications, OData APIs, and Adobe Forms — serving 10+ end users across MM, PP, SD, and FI modules.
  • AI Multi-Agent SDLC Automation: Created AI Agents for the ABAP Process Flow where it receives a Functional Specification (FS) as input and triggers multiple specialized agents — Functional Analysis Agent, Technical Architect Agent, and ABAP Development Agent — which collaboratively produce Effort Estimation (no. of days required), Technical Specification, and the list of ABAP Objects to be developed. Also designed 10+ skill files for AI Copilot (Eclipse) to automate SAP SDLC processes from BRD analysis to development workflows, reducing manual development effort by ~40%
  • Third-Party Application (Ortem) Integration: Developed an end-to-end interface project to integrate third-party applications with SAP, enabling dynamic data exchange. Created Business Objects using RFC framework, built Custom BAPIs, and configured web services via SOAMANAGER ensuring seamless communication and interoperability. Covered multiple interfaces including — Planned Order Creation, Planned Order Deletion, Inventory, Work Center, Bill of Material, Production Order, Routing, and MATMAS.
  • MRP Report Migration (Material Resource Planning): Migrated reporting from SAP Datasphere to SAP Reports for improved integration, functionality, and better performance. Designed custom reports, created supporting tables, and developed Transaction Codes (TCodes). Implemented multi-level reporting (report inside report — 3 reports in total) with dynamic tables to display data according to multi-vendor presence for particular materials. Reduced report generation time by ~50% compared to Datasphere approach.
  • Material Shortage Report (Vehicle Component): Built a report for material shortage detection when vehicle components move between production lines. The report shows Stock, Supplier (component-wise), and Pending GRN so that the responsible MRP Controller is notified of shortages. Previous process: Users manually sent vehicle lists via email, then executed a program limited to 100 vehicles at a time (due to front-end BOM explosion time). After implementation: The new report pulls from the Pending Vehicle List directly (removing email step), processes BOM explosion in background (removing the 100-vehicle limit), and sends output via email with Excel attachment to stakeholders — all without human intervention. Users now save ~1-2 hrs daily
  • Automated Invoice Due Notifications: Automated customer invoice due notifications by creating reports to identify pending dues and send email reminders. Built custom tables and TCodes for efficient execution, improving collection processes and operational efficiency by ~25% across 50+ customer accounts.
  • RAP Objects Development: Designed and delivered 3+ RAP (RESTful ABAP Programming Model) applications using Custom Entities, CDS Views, and Annotations. Implemented Fiori Elements for Excel upload functionality, improving data entry efficiency by ~50% and enabling modern SAP capabilities.
  • Debugging & Technical Support: Resolved 20+ complex technical issues across reports, function modules, classes, enhancement points, OData services, and forms through root-cause analysis and debugging. Delivered technical solutions and implemented program changes to enhance system reliability.
  • Forms & Output Types: Modified 8+ Adobe Forms, SmartForms, and output types to meet evolving business requirements. Enhanced formatting and data accuracy for better document presentation and compliance.
  • API Development: Created 5+ secure and efficient APIs using SAP classes and OData services to enable robust data exchange between SAP and external systems, ensuring seamless interoperability.
